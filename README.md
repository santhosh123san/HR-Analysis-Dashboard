## HR Analysis Dashboard

# Problem Statement
The organization faced challenges in understanding employee data and extracting meaningful
insights for decision-making. The goal of this project was to design a comprehensive HR analytics
dashboard to clearly visualize workforce dynamics such as headcount distribution, trends, and
demographics.

# Tools & Technologies Used
• Excel – Data cleaning and preparation
• Power BI – Data modeling, visualization, and dashboard creation

# Project Workflow
1. Data Collection & Cleaning – HR employee data was prepared and structured.
2. Data Modeling – Relationships and calculated measures were created in Power BI.
3. Data Analysis – Key HR metrics such as headcount, gender distribution, age groups, business
units, job titles, and country-wise analysis were analyzed.
4. Visualization – An interactive HR Analysis Dashboard was built with slicers and KPIs.
5. Reporting – Insights were documented for stakeholders.

# Dashboard Highlights
• Summary KPIs: Total Employees, Male & Female Count
• Headcount trend over time
• Employee distribution by gender, age group, country, business unit, department, and job title
• Interactive slicers for department, country, and employee name
• Data refresh timestamp for reliability

# conclusion
Built an interactive HR dashboard to analyze employee data, headcount trends, and workforce demographics. Used Excel for data cleaning and Power BI for data modeling, KPIs, and visualizations to support data-driven HR decisions.
